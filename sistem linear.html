Sistem Persamaan Linier:#

SISTEM PERSAMAAN LINEAR Ini adalah sekumpulan persamaan yang melibatkan dua atau lebih variabel (biasanya kita pakai x dan y) dan kita ingin menemukan nilai-nilai x dan y yang memenuhi semua persamaan tersebut.

Ada beberapa metode untuk menyelesaikan sistem persamaan linier, salah satunya adalah Eliminasi Gauss. Eliminasi Gauss ini ibarat kita lagi ngerjain soal matematika investigasi. Kita harus pintar-pintar manipulasi persamaan awal untuk mendapatkan solusi.

Pertama, kita harus mengubah persamaan menjadi bentuk matriks.

Matriks itu kayak tabel yang berisi angka-angka sesuai dengan koefisien variabel dan konstanta pada persamaan awal. Setiap baris mewakili satu persamaan.

Kedua, kita ubah matriks tersebut menjadi bentuk eselon baris. Nah, untuk mengubah matriks menjadi bentuk eselon baris, kita bisa melakukan tiga operasi:

  1. Tukar baris: Kita bisa menukar posisi dua baris dalam matriks.

  2. Kalikan baris dengan bilangan: Kita bisa mengalikan semua elemen dalam satu baris dengan bilangan tertentu (asal bukan nol).

  3. Gabungkan baris: Kita bisa menambahkan kelipatan satu baris ke baris lainnya.

Dengan melakukan operasi ini secara cermat, kita bisa melenyapkan variabel tertentu pada baris tertentu.

Selesaikan persamaan berikut ini dengan metode Eliminasi Gaussian $\( 8x_1+9x_2+x_3=19\\ x_1+2x_2+3x_3=9\\ 2x_1+x_2+x_3=5\\ \)$

yang pertama deklarasikan RowSwap, RowAdd dan RowScale untuk perhitungan selanjutnya

def RowSwap(A,k,l):
# =============================================================================
#     A is a NumPy array.  RowSwap will return duplicate array with rows
#     k and l swapped.
# =============================================================================
    m = A.shape[0]  # m is number of rows in A
    n = A.shape[1]  # n is number of columns in A

    B = np.copy(A).astype('float64')

    for j in range(n):
        temp = B[k][j]
        B[k][j] = B[l][j]
        B[l][j] = temp

    return B

def RowScale(A,k,scale):
# =============================================================================
#     A is a NumPy array.  RowScale will return duplicate array with the
#     entries of row k multiplied by scale.
# =============================================================================
    m = A.shape[0]  # m is number of rows in A
    n = A.shape[1]  # n is number of columns in A

    B = np.copy(A).astype('float64')

    for j in range(n):
        B[k][j] *= scale

    return B

def RowAdd(A,k,l,scale):
# =============================================================================
#     A is a numpy array.  RowAdd will return duplicate array with row
#     l modifed.  The new values will be the old values of row l added to
#     the values of row k, multiplied by scale.
# =============================================================================
    m = A.shape[0]  # m is number of rows in A
    n = A.shape[1]  # n is number of columns in A

    B = np.copy(A).astype('float64')

    for j in range(n):
        B[l][j] += B[k][j]*scale

    return B

buat array NumPy dengan nilai-nilai ini. Kita akan memberi nama pada array A, sehingga kita bisa memanggilnya lagi nanti.

import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

# Koefisien untuk masing-masing persamaan bidang
a1, b1, c1, r1 = 1, 2, 3, 9
a2, b2, c2, r2 = 0, 1, 13, 27
a3, b3, c3, r3 = 0, 0, 1, 2

# Buat data x, y
x = np.linspace(-10, 10, 100)
y = np.linspace(-10, 10, 100)

# Buat grid x, y
X, Y = np.meshgrid(x, y)

# Hitung nilai z sesuai dengan persamaan bidang
Z1 = (r1 - a1*X - b1*Y) / c1
Z2 = (r2 - a2*X - b2*Y) / c2
Z3 = (r3 - a3*X - b3*Y) / c3

# Plot persamaan bidang dalam tiga dimensi
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')

# Plot bidang
ax.plot_surface(X, Y, Z1, alpha=0.5, rstride=100, cstride=100, color='r')
ax.plot_surface(X, Y, Z2, alpha=0.5, rstride=100, cstride=100, color='g')
ax.plot_surface(X, Y, Z3, alpha=0.5, rstride=100, cstride=100, color='b')

# Titik perpotongan
A = np.array([[a1, b1, c1], [a2, b2, c2], [a3, b3, c3]])
B = np.array([r1, r2, r3])
intersect = np.linalg.solve(A, B)
ax.scatter(intersect[0], intersect[1], intersect[2], color='black', s=100, label='Titik Perpotongan')

# Label sumbu
ax.set_xlabel('X')
ax.set_ylabel('Y')
ax.set_zlabel('Z')

# Tampilkan legenda
ax.legend()

# Tampilkan plot
plt.show()
